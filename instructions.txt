To run NDM Application

Step 1: Go to this folder -> E:\N.D.M\Programs\N.D.M_Invoice_App_JS

Step 2: Give cmd in search bar and enter it will open the command prompt

Step 3: run this command "net stop MongoDB"

Step 4: run this command "net start MongoDB" for this command we will get the output like below

        The MongoDB Server (MongoDB) service is starting....
        The MongoDB Server (MongoDB) service was started successfully.

Step 5: run "cls" command to clear the screen

Step 6: run this command "nodemon" and we will get the output like below

        [nodemon] 2.0.15
        [nodemon] to restart at any time, enter `rs`
        [nodemon] watching path(s): *.*
        [nodemon] watching extensions: ts,json
        [nodemon] starting `ts-node index.ts`
        Server Running on Port 8080
        MongoDB connected

Step 7: Open the Mongodb Compass and connect the localhost

Step 8: Open the browser and run this url  "http://localhost:8080/index.html" you will landed into N.D.M application

Step 9: After finishing the billing work before shutdown run this command mongodump --out  C:\MogoDB_Backup\mongodb-backup-date
